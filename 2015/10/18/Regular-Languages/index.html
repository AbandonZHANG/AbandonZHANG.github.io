Automata - Regular Language

发表于 Oct 18, 2015

斯坦福大学公开课Automata学习笔记系列（二）。

上次说过，三种有穷自动机（DFA、NFA、ε-NFA）是等价的而且它们能接受的语言都是正则语言。那今天就来说说正则语言。

正则表达式（Regular Expression）

正则语言最常用的表示方式就是正则表达式，它包含两个要素：1.字符（∈字母表Σ）2.操作符。

正则语言的操作有：

  • Concatenation
    两个正则语言L和M串联的结果LM，是L中每个句子（字符串）尾部分别加上M中的每个句子所组成句子的集合。Example：{01,111,10}{00, 01} = {0100, 0101, 11100, 11101, 1000, 1001}。
  • Union
    两个正则语言L和M的并集L∪M，就是L的句子集与M的句子集的并集。Example：{01,111,10}∪{00, 01} = {01,111,10,00}。
  • Kleene Star
    正则语言L的克莱尼星L*是将0个L串联、1个L串联、……、n个L串联结果的并集。Example：{0,10}* = {ε, 0, 10, 00, 010, 100, 1010, …}。

如果E1、E2是两个正则表达式，它们表示的正则语言分别是L(E1)和L(E2)，则有：

  • L(E1+E2) = L(E1)∪L(E2)
  • L(E1E2) = L(E1)L(E2)
  • L(E1*) = L(E1)*

正则语言的特性

  • 闭包（Closure）
    • 任意两个正则语言的Concatenation、Union、Kleene Star操作的结果仍然是正则语言。
    • 正则语言的差集L-M仍然是正则语言。
    • 正则语言的补集Σ*-L仍然是正则语言。
    • 正则语言的同构（映射关系h(0)=ab）仍然是正则语言。
  • 单位元和零元：
    • ∅是Union操作的单位元。即R+∅=R。
    • ε是Concatenation操作的单位元。即Rε=εR=R。
    • ∅是Concatenation操作的零元。即R∅=∅。
  • 判断句子（字符串）是否属于正则语言L：
    • 判断句子是否被表示正则语言L的DFA所接受。
  • 判断一个正则语言是否是空集：
    • 判断表示正则语言的DFA的终止状态是否可达。
  • 判断一个正则语言是否是一个无穷集合：
    • 如果表示正则语言的DFA有n个状态，而该正则语言有长度大于n的句子，则该正则语言一定是无穷集合。（有环）
  • 泵引理：
    • 对于任何一个正则语言L，都存在一个整数n（–> DFA状态数），对于任何一个长度大于n的L的字符串，都可以写成w=xyz的形式，其中：
      • |xy| < n
      • |y| > 0
      • 所有的i >= 0，xy^iz都属于这个正则语言
  • 判断两个正则语言L和M是否相等 –> 判断表示L和M的DFA是否相等：
    • 构造一个新的DFA，其状态集是L和M的状态集的笛卡尔积(q,r)
    • δ([q,p], a) = [δ_L(q,a), δ_M(p,a)]
    • 对于一个新DFA的状态[q,p]，如果q和p其中有且仅有一个是L或M的终止状态，则[q,p]是新DFA的终止状态
    • L和M相等 –> 新的DFA的终止状态不可达
  • “最小化”DFA –> 去掉“相同”的状态
    • 对于DFA的任意两个状态，如果其中有且仅有一个状态，则这两个状态是不同的
    • 对于任意的状态q和p，如果δ(q,a)和δ(p,a)不同，则q和p不同
    • 合并剩下“相同”的状态
    • 去掉不可达的状态

正则表达式与FA的等价与转换

（一）正则表达式可以很方便地构造出对应的ε-NFA：

  • Concatenation
  • Union
  • Kleene Star

（二）DFA向正则表达式转换有一点点难，需要一点动态规划和归纳的思想，接触过Floyd算法的理解起来会比较方便。算法描述如下：

  • Desc：将DFA的状态标为（1, …, n），D(i,j,k)表示从DFA的状态i到状态j，中间只能经过不大于k的状态所表示的正则表达式。
  • Base：k=0，D(i,j,0) = union{从状态i转移到状态j的字符}
  • Induction：
    • 假设当前所有小于k的D(i,j,)表达式都已经算出
    • D(i,j,k) = D(i,j,k-1) + D(i,k,k-1)(D(k,k,k-1))*D(k,j,k-1)}
  • Final：DFA所表示的正则表达式=Union{D(i,j,n)|i=起始状态，j∈终止状态，n=DFA状态数}
