META-INF 和 WEB-INF

发表于 Jan 28, 2017

WEB-INF

WEB-INF 是 WAR（Web Application Archive）包的一个目录，维护一些 Web 服务端不想让客户端通过URL访问到的资源文件，比如：web.xml，以及 Java 的 classes 和 library 文件等。

META-INF

META-INF 是 JAR（Java Archive File）包的一个目录，而 JAR 包本质上就是 zip 格式的压缩包加上一个 META-INF 目录。

META-INF 目录下存放 Java 应用程序的一些元信息（To configure applications, extensions, class loaders and services）。一个标准的 META-INF 目录通常都包含 MANIFEST.MF 、INDEX.LIST、*.SF、*.DSA、services/ 等文件和目录，此外也可以放一些用户自己的文件。

由于 WAR 也是一个 JAR 文件，因此 WAR 包下也有 META-INF 目录。
