JVM 参数

发表于 Dec 04, 2017

标准参数

选项 说明
-server 以 server 模式运行

-X 参数

选项 说明
-Xmx3g 设置堆最大空间
-Xms3g 设置堆初始空间

-XX 参数

-XX:+DisableExplicitGC：System.gc() 变成空调用

CMS 垃圾回收器

-XX:+UseConcMarkSweepGC：使用 CMS 垃圾回收器（新生代默认使用 ParNew）

-XX:+ExplicitGCInvokesConcurrent：System.gc() 时使用 CMS gc 替换 Full gc

-XX:+ExplicitGCInvokesConcurrentAndUnloadsClasses：System.gc() 时使用 CMS gc 替换 Full gc，并且方法区（Perm）也在回收范围内

-XX:CMSInitiatingOccupancyFraction=80%：CMS 第一次 gc 时参考的 old 区空间占用比例

-XX:+CMSInitiatingOccupancyOnly：强制 JVM 不基于运行时收集的数据动态调整 gc 时空间占比阈值，每次都使用 CMSInitiatingOccupancyFraction 的值

G1 垃圾回收器

-XX:+UseG1GC：使用 G1 垃圾回收器（新生代&&老年代）
